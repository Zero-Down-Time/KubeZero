# ![k8s-v1.33](images/k8s-1.33.png)  KubeZero 1.33 - Octarine

## What's new - Major themes
- Alpine OS upgrade to V3.22
- Migrated all iptables configs to nftables for Bastion, NAT, VPN GWs etc.
- new local firewall rules for all EC2 instances as additional layer, rate limiting for SSH logins etc
- Redis Operator - migrated off Bitnami images
- Strimzi Kafka Operator
- CloudnativePG Operator

## Features and fixes
- improved NAT instance memory config allowing to run on t3a.nano
- controller nodes run on 8GB memory t3a.large by default to improve latency
- various enhancements to run KubeZero on GCP/GKE
